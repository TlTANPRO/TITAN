## Bug Fixes

### 1. `scripts/validate-merge.mjs` line 120

**Bug**: `f.endsWith('.json')` matched `*.backup-2026-07-19.json` → cross-file dedup treat backup as duplicate → all posts deleted from current scraped.

**Dampak**: deploy `accounts-full.json` 0 post (commit e763940, 19 Jul 2026). Recovered ke V20 commit bc22828.

**Fix**:
```js
// Sebelum:
const files = (await fs.readdir(SCRAPED_DIR)).filter((f) => f.endsWith('.json'));
// Sesudah:
const files = (await fs.readdir(SCRAPED_DIR)).filter((f) => f.endsWith('.json') && !f.includes('.backup-'));
```
